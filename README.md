# 진로탐색프로젝트 - 알고리즘과 생명과학
이 코드는 격자에서 치즈(=세포 덩어리) 주변을 BFS로 탐색하여 외부 공기와 접촉한 면의 수를 계산하고, 동일한 양의 치즈가 한 덩어리일 때와 여러 덩어리일 때 표면적/부피 비율의 차이를 비교한다. 한 덩어리는 외부와 닿는 면이 적어 내부까지 산소·영양분이 확산되기 어렵고, 두 덩어리는 표면적이 상대적으로 커져 물질 교환이 훨씬 효율적이다. 이는 실제 세포가 커지기만 하면 내부까지 확산이 충분히 이루어지지 않아 생존에 불리하다는 점을 반영한다. 따라서 생명체가 일정 크기 이상에서 분열을 통해 더 작은 세포 여러 개로 유지되는 이유를 이해할 수 있다.
